Durante este curso aprendí que la tecnología puede facilitar la participación ciudadana y mejorar la organización de las comunidades. Comprendí la importancia de integrar una base de datos, un servidor y herramientas de automatización para crear soluciones útiles. También entendí cómo generar documentos automáticamente para agilizar procesos. Aprendí que la protección de los datos personales es fundamental para brindar seguridad y confianza a los usuarios. Además, conocí el uso de herramientas como Git y GitHub para organizar y compartir proyectos. El curso me permitió desarrollar habilidades prácticas para crear aplicaciones sencillas. Considero que estos conocimientos serán útiles en futuros proyectos académicos y profesionales relacionados con el desarrollo de software y la participación comunitaria.
